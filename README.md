# File Management System

## 📌 Description
A Python's OOPs based file management system to create, read, update, and delete files.

## 🚀 Features
- Create new files
- Read file contents
- Update existing files
- Delete files

## 🛠️ Technologies Used
- Python
- OOPs

## ▶️ How to Run
python main.py
